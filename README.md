# Windows-Boot-Logo-Changer
一款跨平台支持的 Windows 开机图标自定义替换工具，无需复杂手动操作，通过脚本一键实现开机图标的个性化修改，适配 Windows 7/8.1/10/11 系统。

## 🔍 项目核心用途
1. 替换 Windows 系统默认开机图标（包括启动时的品牌Logo、进度条样式关联图标等）；
2. 支持自定义图标格式导入，兼容主流图片格式（需符合 Windows 开机图标分辨率要求）；
3. 跨平台辅助部署：Windows/macOS/Linux 系统均可通过对应脚本完成工具配置或打包，适配不同用户的开发/使用环境。

## 🚀 功能特性
- ✅ 多系统适配：工具运行依赖 Windows 系统（目标修改系统），配置/打包支持 Windows/macOS/Linux；
- ✅ 一键操作：通过脚本自动化完成图标替换流程，无需手动修改系统核心文件；
- ✅ 安全兼容：适配 Windows 主流版本，避免因系统版本差异导致的替换失败；
- ✅ 可打包部署：提供打包配置文件，支持将工具编译为独立可执行程序，方便分发使用。

## 📋 使用步骤（Windows 系统下替换图标）
### 前提条件
- 目标系统：Windows 7/8.1/10/11（32/64位均可）；
- 权限要求：运行脚本时需右键选择「以管理员身份运行」；
- 图标准备：提前准备符合要求的图标文件（建议分辨率：1024×1024，格式：.png/.bmp，需无背景透明）。

### 操作流程
1. 下载本仓库所有文件，解压至本地（建议路径无中文、无空格，如 `D:\Tools\Windows-Boot-Logo-Changer`）；
2. 将自定义图标文件命名为 `custom_logo.png`，放入 `scr/resource/` 目录（若无该目录可手动创建）；
3. 右键点击 `Windows开机图标替换工具.bat`，选择「以管理员身份运行」；
4. 按照脚本提示输入对应选项（如确认图标路径、选择目标系统版本），等待脚本执行完成；
5. 执行成功后，重启电脑即可看到新的开机图标。

## ⚠️ 注意事项
1. 操作前请备份系统关键文件（或创建系统还原点），避免因误操作导致开机故障；
2. 部分 Windows 10/11 系统需关闭「安全启动」才能生效（BIOS/UEFI 中设置）；
3. 若替换后开机异常，可重新运行脚本选择「恢复默认图标」，或通过系统还原点还原；
4. 工具仅用于个人学习和个性化需求，请勿用于商业用途或未经授权的设备修改。

## 🛠️ 打包编译教程（开发者用）
### 1. Windows 下打包为可执行程序
- 安装依赖：`pip install pyinstaller inno-setup`；
- 运行打包脚本：`pyinstaller Windows开机图标替换工具.spec`；
- 生成安装包：打开 Inno Setup，加载 `setup.iss`，点击「编译」生成安装程序。

### 2. macOS/Linux 下配置环境
- 安装依赖：`pip install pyinstaller`；
- 运行辅助脚本：`chmod +x Windows开机图标替换工具.command`（macOS）或 `chmod +x Windows开机图标替换工具.sh`（Linux）；
- 执行脚本：`./Windows开机图标替换工具.command`（macOS）或 `./Windows开机图标替换工具.sh`（Linux），完成环境配置。
